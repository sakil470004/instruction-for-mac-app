# YouTube Download by Terminal

## Basic Download Command

```bash
yt-dlp "YouTube link"
```

## Download Best Quality

```bash
yt-dlp "YouTube link" -S res,ext:mp4:m4a --recode mp4
```

## ✅ Supported Sites

- YouTube
- Facebook
- Twitter / X
- Instagram
- TikTok
- Vimeo
- Dailymotion
- Bilibili
- SoundCloud
- Many news sites, adult sites, and streaming services (depending on DRM or login requirements)
